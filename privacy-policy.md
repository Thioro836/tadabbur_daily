# Politique de Confidentialité — Tadabbur Daily

Dernière mise à jour : avril 2025  
Développeur : Thioro BAH

## Données collectées
Tadabbur Daily ne collecte aucune donnée personnelle.
Toutes vos données (journal, favoris, préférences) sont
stockées uniquement sur votre appareil.

## Données non partagées
Aucune donnée n'est transmise à des serveurs externes
ni partagée avec des tiers.

## API utilisée
L'app utilise l'API Al-Quran Cloud (alquran.cloud)
uniquement pour récupérer les versets et l'audio.
Aucune donnée personnelle n'est envoyée à cette API.

## Notifications
Les notifications sont gérées localement sur votre
appareil. Aucune donnée n'est collectée.
